# Inventory Management System

A simple Java console application for managing inventory.

## Features

* Add products
* Remove products
* Search products
* Display all products
* Calculate total inventory value

## Technologies

* Java
* ArrayList
* Object-Oriented Programming (OOP)

## Classes

* Product
* Inventory
* Main

## Example

1. Add Product
2. Remove Product
3. Search Product
4. Show Products
5. Total Inventory Value
6. Exit
